# Blogging App Frontend

React frontend for the Spring Boot Blogging App backend.

## Tech stack
- React 18
- Vite 5

## Run locally
1. Start the backend on port 8080.
2. In this folder run:
   - npm install
   - npm run dev
3. Open the Vite URL shown in the terminal.

The Vite dev server proxies /v1 requests to http://localhost:8080.

## Build
- npm run build
- npm run preview
